ESP32 Wireless ROV Control System

In this project, I developed an ESP32-based system to control a remotely operated vehicle (ROV). The system consists of two ESP32 modules: one on the ROV and another in a handheld mobile station. Using NRF24L01+ radio modules, the mobile station establishes a wireless link to an external base, enabling remote control of the ROV. This setup allows real-time transmission of commands and telemetry, ensuring precise and effective surface-level operation.

Door Lock System

The system consists of two microcontrollers: HMI (Human-Machine Interface) and ECU (Electronic Control Unit). The HMI is connected to an LCD, keypad, and reset push button, while the ECU is connected to an EEPROM, motor driver, and buzzer. Users can set and save a password in the EEPROM, with a maximum length of 5 characters. The system allows password changes when logged in, ensuring secure access management.

WhatsAutoX

WhatsAutoX automates sending WhatsApp messages via the command line using WhatsApp Web. The tool utilizes image recognition for status checks, ensuring the correct login state and handling invalid numbers. It supports automated messaging for multiple recipients, flexible input methods (text files or direct command-line input), and is compatible with both Windows and Linux.
